### tetris with game Library "sfml"

###### Requires sfml ( version 2.4.2 ) : Link https://www.sfml-dev.org/download/sfml/2.4.2/

+ features
  - show next piece
  - fast drop 
  - piece fall
  - predict drop location
  - "shuffle bag" for random piece
  
+ button 
  - reft, right arrow : move
  - up arrow : rotate
  - down arrow : fast drop
  - space : piece fall
